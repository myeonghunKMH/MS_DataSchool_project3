📘 서비스 사용 가이드

암호화폐 모의 투자 서비스를 안전하게, 빠르게 익히기 위한 안내서입니다.
아래 섹션을 차례대로 살펴보고 바로 실습으로 넘어가 보세요.

🚀 서비스 소개

암호화폐 시장은 24시간 거래·높은 변동성·규제 불확실성 등으로 초보자에게 진입장벽이 큽니다.
본 서비스는 이러한 환경에서 안전하게 연습하고 시장 이해를 돕도록 설계되었습니다.

실제 데이터를 활용한 시뮬레이션부터 실시간 뉴스, 교육 자료, 모의 투자까지
암호화폐 투자에 필요한 모든 기능을 제공합니다.

1️⃣ 시뮬레이션

  • 과거의 실제 암호화폐 거래 데이터를 기반으로 시나리오별 시뮬레이션 제공
  • 중요 사건이 발생했던 시점을 불러와 체험
  • 사용자가 투자금·매수/매도 결정을 직접 선택
  • 실제 과거 데이터 흐름으로 결과 확인 → "그때 투자했다면?" 학습
💡 과거 사건의 시장 영향과, 나의 의사결정이 결과에 미치는 효과를 체험합니다.

2️⃣ 뉴스 한눈에 보기

  • 네이버에서 실시간으로 크롤링한 가상화폐 관련 최신 뉴스 제공
  • 시장에 영향을 미치는 주요 이슈를 한 곳에서 빠르게 확인
  • 카테고리별로 정리된 뉴스로 효율적인 정보 수집
  • 투자 결정에 필요한 시장 동향과 트렌드를 실시간으로 파악
📰 최신 뉴스와 시장 정보를 놓치지 않고 투자 전략에 활용하세요.

3️⃣ 암호화폐 백과사전

  • 대표 암호화폐들의 특징과 기술적 배경 상세 설명
  • 블록체인, 채굴, 지갑 등 핵심 개념 정리
  • 시가총액, RSI, 이동평균선 등 투자 지표 해설
  • 초보자도 이해하기 쉬운 체계적인 학습 자료 제공
📚 암호화폐 투자에 필요한 기초 지식부터 심화 내용까지 체계적으로 학습하세요.

주요 코인 정보

비트코인, 이더리움 등 주요 암호화폐의 특징과 활용 사례

📊 투자 지표 설명

차트 분석과 투자 판단에 필요한 각종 지표들의 의미와 활용법

🔍 개념 이해

블록체인 기술과 암호화폐 생태계의 핵심 개념들

4️⃣ 실시간 모의 투자 연습

  • 업비트의 실시간 데이터를 기반으로 한 실전 모의 투자
  • 실제 거래소와 동일한 인터페이스와 기능 제공
  • 가상 자금으로 리스크 없이 투자 전략 테스트
  • 주문/체결/잔고 관리 등 실전과 동일한 거래 경험
  • 나의 거래 기록과 수익률을 통한 투자 성과 분석
💰 실제 자금 투자 전, 충분한 연습으로 자신만의 투자 전략을 완성하세요.

5️⃣ 나의 투자 보고서

  • 모든 거래 히스토리를 바탕으로 한 상세 투자 분석 리포트
  • 수익률, 승률, 주요 거래 패턴 등 투자 성향 분석
  • 개선이 필요한 투자 습관과 맞춤형 조언 제공
  • 실제 투자 전 나의 강점과 약점을 파악하여 투자 전략 개선
📈 데이터 기반의 객관적인 분석으로 더 나은 투자자로 성장하세요.
